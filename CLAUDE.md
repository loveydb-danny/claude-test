# CLAUDE.md

이 파일은 Claude Code (claude.ai/code)가 이 저장소에서 작업할 때 참고하는 가이드입니다.

## 프로젝트 개요

유명한 명언을 보여주는 간단한 웹페이지입니다. HTML, CSS, JavaScript만으로 구성된 정적 단일 페이지 애플리케이션입니다.

## 프로젝트 구조

- `index.html` - HTML 구조, CSS 스타일, JavaScript 로직이 모두 포함된 단일 파일

## 아키텍처

### 단일 파일 구조
모든 코드가 `index.html` 안에 포함되어 있습니다:
- **스타일** (7-158줄): `<style>` 태그 내부, 그라데이션 배경, 애니메이션, 반응형 디자인 포함
- **HTML 구조** (161-172줄): 명언 텍스트, 작가, 버튼 요소를 담은 컨테이너
- **JavaScript 로직** (174-278줄): 명언 데이터 배열 및 랜덤 선택 로직

### 데이터 구조
`quotes` 배열 (175-256줄)에는 `text`와 `author` 속성을 가진 20개의 명언 객체가 포함되어 있습니다. `getNewQuote()` 함수를 통해 이전과 중복되지 않게 랜덤으로 표시됩니다.

### 주요 함수
- `getNewQuote()` (260-272줄): 현재와 다른 명언을 랜덤으로 선택하고 DOM 업데이트
- `window.onload` (275-277줄): 페이지 로드 시 첫 명언을 랜덤으로 표시

## 개발 방법

### 로컬 개발
브라우저에서 `index.html`을 직접 열거나, 로컬 서버를 실행하세요:

```bash
python3 -m http.server 8000
```

그 다음 브라우저에서 `http://localhost:8000` 접속

### 커스터마이징

**명언 추가/수정**: `<script>` 섹션의 `quotes` 배열 수정 (175-256줄)

**스타일 변경**:
- 주요 그라데이션 색상: `#667eea` 와 `#764ba2` (16줄, 102줄)
- 모바일 반응형 기준점: 768px (129줄)
- 컨테이너 최대 너비: 800px (28줄)

**언어 변경**: 현재 한국어로 작성됨. 다른 언어로 변경하려면:
- HTML `lang` 속성 수정 (2줄)
- HTML 및 JavaScript의 모든 텍스트 내용 변경
- 페이지 제목 변경 (6줄)
